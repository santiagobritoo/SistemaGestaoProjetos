# Sistema de Gestão de Projetos

## Sobre o projeto

Este projeto foi desenvolvido em Java com o objetivo de praticar Programação Orientada a Objetos (POO).

O sistema permite organizar informações de usuários, projetos, equipes e tarefas.

## Funcionalidades

- Cadastro de usuários
- Cadastro de projetos
- Cadastro de equipes
- Cadastro de tarefas

## Tecnologias utilizadas

- Java
- Eclipse
- GitHub

## Como executar

1. Abrir o projeto no Eclipse.
2. Executar a classe Main.java.
3. Utilizar as opções do menu.
